# EuskoMove
Plataforma web para consultar horarios, reservar asientos y comprar billetes de autobús del País Vasco.

## ¿Qué incluye?

- Consulta de horarios y líneas
- Reserva de asientos con mapa visual del autobús
- Compra de billetes con pago seguro (HTTPS / TLS 1.2)
- Panel de avisos y noticias (solo admin)
- Login y registro de usuarios

## Instalación y arranque

```bash
pip install flask
python app.py
```

Abre: http://127.0.0.1:3000
